**Doctrine (DBAL, ORM) tasks for Robo Task Runner.**


## Install

Via Composer

``` bash
$ composer require indigophp/robo-doctrine
```


## Usage

1. Use one of the `loadTasks` traits in your `RoboFile`
2. Run commands as usual
